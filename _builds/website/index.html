About Info

O aplikacji About

MójPokedex to Pokedex dla Pokemon GO. Oprócz samej listy pokemnów dostępne są też listy ataków danego pokemona wraz DPS (damage per second, czyli obrażenia na sekundę).

Wersja aplikacji 0.3.1.

MyPokedex is a Pokedex for Pokemon GO. It includes available attacks and DPS (damage per second) calculation for each pokemon.

App version 0.3.1.

*) DPS (damage per second) — note that the column inclundes bonus for attacks of the same type as the pokemon, so called STAB (same type attack bonus)

Markings before attack name
☒ attack is no longer availble in the wild,
☑ attack can appear in newly acquired pokemon.
(more on fevgames).
